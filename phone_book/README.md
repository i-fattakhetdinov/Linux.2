# Phone book
### Сборка и запуск
Для запуска необходимо выполнить
```bash
make
make mount
make run
```
### Доступные команды
+ Добавление записи
```bash
add first_name last_name age phone_number email
```
+ Получение информации по фамилии
```bash
get last_name
```
+ Удаление записей с данной фамилией
```bash
del last_name
```
